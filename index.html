Disease Warner for Google Map

ETCE 是 2015/10/6 參與臺泰跨國黑客競賽的作品
目標是結合 Google Map API
當使用者規劃之旅程經過疫情嚴重區域時
顯示警示資訊
此想法的價值在於不要求使用者額外下載 APP
不需要使用者特別記得做旅程規劃時要查詢疫區資訊
並且透過將資料轉為 geoJSON 格式
可將應用範圍擴大至急難避災、交通管理等

Handwriting.js

handwriting.js 是介接 google 手寫辨識 API 的捷徑
透過預先定義之 handwriting 物件
包含常用之復原、重做及預測結果過濾
讓 Web App 實作手寫相關功能的門檻大幅降低

Font Embedding

中文字成千上萬，要做出完整的手寫字型相當困難
所以在製作字型時會依據情況只寫缺少的字
然而當資料庫漸趨龐大
肥大的字型檔案會造成頁面載入延遲
於是相對應的
只向資料庫要求需要的字
可以大幅增加速度
